SHABR

This project was to see if we could use our new bootstrap skills to create a website. We had to use bootstrap website to learn new tools that we could implement into our website. Then we had to use the grid system to know where each div class goes, so our websites don't look a mess. It is our first ever created website, and we used bootstrap to help us.


Screenshot of Website


I tried to have an even spacing throughout the website. I wanted to have the carousel of images be the first thing in sight, so when a person goes on the website it catches their eye. Then after that I used a jumbotron to display the home. I used badges to identify the info. I used buttons and cards to display the testimonials. With all of these tools helped me make a pretty good first website. The bootstrap website explained all of the tools very well.
